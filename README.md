# volery-client
This is our conference app: Volery!

## Testing

Tests have been setup according to this https://docs.expo.dev/develop/unit-testing/#install-react-native-testing-library. To run the tests just do `cd expo-frontend && npm run test` in the root dir. You should see 1 test pass. More information is available here - https://nten.atlassian.net/wiki/spaces/Volery/pages/15532033/Testing.
